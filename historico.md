# Albedrío Security - Bitácora de Desarrollo

## Filosofía del Proyecto
El sistema Albedrío Security nace como una respuesta a la necesidad de soberanía digital total. No es solo un antivirus o un firewall; es una **extensión del hardware** del usuario. La premisa es simple: si el silicio no te reconoce, el sistema no existe para ti.

## Leyes del Sistema
1. El sistema solo obedece a la presencia física del dueño.
2. Cualquier intento de intrusión remota dispara la respuesta Petaboom.
3. El Kernel es el ejecutor final; no hay bypass por software.
